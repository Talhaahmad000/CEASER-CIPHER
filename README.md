# Caesar Cipher

A simple Python implementation of the Caesar Cipher algorithm that allows users to encrypt and decrypt messages using a custom shift key.

## Features

* Encrypt text using a shift key
* Decrypt the encrypted text
* Interactive command-line interface
* Supports continuous execution until the user exits

## How It Works

The Caesar Cipher is a substitution cipher where each letter in the message is shifted by a fixed number of positions in the alphabet.

### Example

Message: `hello`

Shift Key: `3`

Encrypted Text: `khoor`
